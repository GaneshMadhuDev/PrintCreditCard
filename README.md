# PrintCreditCard
An Angular 1.x web application for form the credit card / validating card details / printing credit card


Functional

1. Add Credit Card Details
2. Card Details : 
    Card Number,
    Card Holder,
    Card Exp Date,
    Card CVV
    
3. Flipping Card to other side
4. Print Credit Card
5. Credit Card Number Validations

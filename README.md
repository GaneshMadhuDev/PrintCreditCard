# PrintCreditCard
An Angular 1.x web application for form the credit card / validating card details / printing credit card


Functional

1. Added Credit Cards
2. Card Details : 
    Card Number
    Card Holder
    Card Exp Date
    Card CVV
    
3. Flipping Card
4. Print Credit Card
